# Study Practice Helper

一个本地运行的浏览器题库整理与复习插件，支持学习通题库采集、刷题、错题、回忆题、考频分析和文档导出。

本项目面向课程复习、题库整理和个人学习分析。插件数据默认保存在浏览器本地，不读取账号密码，不向外上传题库。

## 功能概览

- 从当前页面读取并整理题目
- 支持整卷预览批量提取题目、答案和解析
- 新增**自测极速批量采集**方案，单次最高可一次性抓取500道带完整答案试题，无需逐题翻页等待
- 支持单选、多选、判断、填空、主观题
- 支持 JSON、普通文本、学习通页面文本导入
- 支持课程、学期、来源等元数据
- 支持回忆题导入，允许只记录关键词、考点、选项或答案
- 支持全部题、错题、收藏题、高频 Top100、知识点专项练习
- 支持输入题号精准跳转和随机下一题
- 支持自动判题、显示答案解析、收藏、重置进度
- 支持题型分布、知识点/标签、高频关键词、考频统计
- 支持导出题库文档、分析报告和高频题 Top100

## 安装

1. 下载或克隆本仓库。
2. 打开 Chrome 或 Edge。
3. 进入扩展程序页面。
4. 打开“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择本项目目录。

## 使用流程

```text
采集题库 -> 导入/补充回忆题 -> 分析考频 -> 高频专项练习 -> 导出复习文档
```

### 采集题库

在学习通题库页或整卷预览页打开插件：

#### 基础采集功能

- 点“读取并整理”可把当前页面内容整理为标准题库 JSON。
- 点“保存当前题”可把当前题直接加入本地题库。
- 点“保存并下一题”可保存当前题并跳到下一题。
- 点“开始连续采集”可连续保存题目，支持设置数量和中途停止。

#### 大批量极速采集（500题高效方案）

1. 在学习通创建自测试卷，配置好全部需要采集的试题；
2. 直接提交自测，页面会一次性展示整套试卷**全部题目+完整标准答案**；
3. 此时点击插件「读取并整理」一键抓取，单次最高可获取500道完整试题，省去逐题翻页等待，适合大批量题库快速录入。

### 刷题练习

插件支持以下练习方式：

- 全部题目
- 错题模式
- 收藏题模式
- 高频 Top100
- 点击知识点/标签/关键词后的专项练习

### 题库分析

题库分析面板会统计：

- 总题量
- 答案覆盖率
- 已练习数量
- 错题数量
- 回忆题数量
- 题型分布
- 知识点/标签分布
- 高频关键词
- 高频题 Top10

高频题会综合完全重复题、相似题、同知识点题、知识点热度、关键词热度和回忆题可信度来排序。

## 支持的数据格式

### JSON 题库

```json
[
  {
    "course": "马克思主义基本原理",
    "semester": "2026春",
    "source": "学习通题库",
    "type": "single",
    "question": "下列哪一项属于操作系统？",
    "options": ["A.Windows", "B.Word", "C.Excel", "D.PowerPoint"],
    "answer": ["A.Windows"],
    "analysis": "Windows 是操作系统。",
    "tags": ["计算机基础"]
  }
]
```

题型可选：

- `single`：单选
- `multiple`：多选
- `judge`：判断
- `blank`：填空
- `text`：主观题

也兼容中文题型：`单选题`、`多选题`、`判断题`、`填空题`、`主观题`。

### 普通文本

```text
1. 下列哪一项属于操作系统？
A. Windows
B. Word
C. Excel
D. PowerPoint
答案：A
解析：Windows 是操作系统。

2. 以下属于输入设备的是？
A. 键盘
B. 显示器
C. 鼠标
D. 打印机
答案：A、C
解析：键盘和鼠标用于输入信息。
```

### 学习通页面文本

```text
1. (单选题)
在实际工作中，要注意掌握分寸，防止“过”或“不及”，其关键在于（）。
A
抓住事物的主要矛盾
B
确定事物的质
C
认识事物的量
D
把握事物的度
答案：D
```

插件会保留选项编号，并把 `答案：D` 转换成完整答案，如 `D.把握事物的度`。

### 回忆题

回忆题允许不完整，每题之间空一行：

```text
考点：认识论 实践
只记得好像考了实践是认识的基础
答案：实践

考点：资本主义 经济危机
生产过剩相关选择题
```

## 导出内容

插件支持导出：

- 题库 Word 文档
- 题库分析 Word 文档
- 高频题 Top100 Word 文档

题库文档开头会包含分析摘要，包括主要知识点、高频关键词和题型分布。

## 隐私与使用边界

- 插件数据保存在浏览器本地。
- 插件不会读取账号密码。
- 插件不会把题库上传到服务器。
- 请不要把含有姓名、学号、课程内部资料或未获授权的题库上传到公开仓库。
- 本项目用于学习复习、题库整理和个人学习分析，请遵守学校、课程和平台规则。
