# Contributing

欢迎改进这个项目。提交代码前请注意：

1. 不要提交真实题库、个人姓名、学号、课程内部资料或账号信息。
2. 尽量保持插件本地运行，不引入不必要的网络请求。
3. 修改解析逻辑时，请同时补充 README 中的格式说明。
4. 修改界面后，请在 Chrome 或 Edge 中重新加载扩展并手动检查主要流程。

## 建议的提交内容

- 更稳的题目解析规则
- 更清晰的导入/导出格式
- 更好的题库分析算法
- 更好的本地刷题体验
- 文档和使用说明改进

## 不建议提交

- 自动答题、自动提交或绕过平台限制的功能
- 包含个人信息的截图或示例数据
- 与题库整理和复习无关的大型依赖
