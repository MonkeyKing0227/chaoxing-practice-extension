# Usage Guide

## 加载扩展

1. 打开 Chrome 或 Edge 的扩展程序页面。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择项目根目录。

## 导入题库

可以使用三种方式：

- 在学习通页面点击“读取并整理”。
- 在管理题库中粘贴 JSON 或普通文本并点击“追加导入”。
- 在整卷预览页面使用“读取并整理”批量提取。

## 刷题

- 点“全部”刷完整题库。
- 点“错题”刷做错过的题。
- 点“收藏”刷收藏题。
- 点“刷高频Top100”刷高频题。
- 点击知识点或关键词后，可进入专项练习。

## 导出

在“管理题库”中可以导出：

- 完整题库文档
- 高频题 Top100

在“题库分析”中可以导出：

- 分析报告
